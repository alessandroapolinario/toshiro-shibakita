Docker: Utilização prática no cenário de Microsserviços

Muito se tem falado de containers e consequentemente do Docker no ambiente de desenvolvimento. Mas qual a real função de um container no cenários de microsserviços? Qual a real função e quais exemplos práticos podem ser aplicados no dia a dia? Essas são algumas das questões que serão abordadas de forma prática neste projeto. Pré-requisitos: Conhecimentos básicos em Linux, Docker e AWS.
